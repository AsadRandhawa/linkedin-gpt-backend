# Privacy Policy for LinkedIn GPT Comment Suggester

**Last updated:** January 28, 2026

## Overview
LinkedIn GPT Comment Suggester is a Chrome extension that helps users generate AI-powered comment suggestions for LinkedIn posts.

## Information We Collect

### What We Collect
- **LinkedIn Post Content**: Only when you click the "✨ AI Reply" button, the text content of the LinkedIn post is sent to our backend API.

### What We DON'T Collect
- No personal information
- No LinkedIn credentials
- No browsing history
- No cookies
- No user profiles
- No data when you're not actively using the extension

## How We Use Your Information

### Processing
- Post content is sent to our secure backend API hosted on Railway.app
- Content is processed by OpenAI's GPT-4 API to generate comment suggestions
- Processing happens in real-time and only when you click "AI Reply"

### Storage
- **No permanent storage**: We do not store any post content or generated suggestions
- **No logs**: We do not log user data or activity
- **No database**: We do not maintain a database of user information

## Data Sharing

### Third-Party Services
We use the following third-party services:

1. **OpenAI GPT-4 API**
   - Purpose: Generate comment suggestions
   - Data shared: LinkedIn post content only
   - Privacy policy: https://openai.com/privacy

2. **Railway.app**
   - Purpose: Backend API hosting
   - Data shared: LinkedIn post content (in transit only)
   - Privacy policy: https://railway.app/legal/privacy

### No Other Sharing
- We do not sell your data
- We do not share data with advertisers
- We do not use data for marketing purposes

## Data Security

### Encryption
- All data transmission uses HTTPS encryption
- Communication between extension and backend is secure

### Access Control
- Only our backend API has access to post content during processing
- No human access to user data
- No data retention after processing

## Your Rights

### Control
- You control when data is sent (only when you click "AI Reply")
- You can stop using the extension at any time
- Uninstalling removes all extension data from your browser

### No Account Required
- No registration or account creation required
- No email collection
- No user tracking

## Permissions Explanation

### Why We Need Permissions

**activeTab**
- Required to read LinkedIn post content when you click "AI Reply"
- Only active on LinkedIn.com

**scripting**
- Required to inject the "✨ AI Reply" button into LinkedIn pages
- Only runs on LinkedIn.com

**host_permissions (Railway.app)**
- Required to communicate with our backend API
- Only used when generating suggestions

## Children's Privacy
This extension is not directed at children under 13. We do not knowingly collect information from children.
